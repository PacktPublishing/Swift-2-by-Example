Swift 2 by Example

This is the code repository for Swift 2 by Example, published by Packt Publishing. It contains all the supporting project files necessary to work through the book from start to finish.

Instructions and Navigation

All of the code is organized into folders. Each folder starts with number followed by the application name. For example, 1-GuessTheNumber which is application from chapter 1.

Each Chapter contains lots of code and many swift files mainly AppDelegate.swift and ViewController.swift. You will see code something similler to the following:

import UIKit

        return true
    }
}
